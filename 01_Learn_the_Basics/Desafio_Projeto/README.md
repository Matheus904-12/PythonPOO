## 🚀 Projeto: NEXUS - Gerenciador de Colaboradores

O NEXUS é uma ferramenta CLI (Linha de Comando) para cadastrar e analisar uma equipe pequena.

### 📋 Requisitos do Sistema

1. **Menu Principal**: O sistema deve rodar em um loop (`while True`) mostrando as opções:
   - [1] Cadastrar Colaborador
   - [2] Exibir Relatório da Equipe
   - [3] Sair
2. **Cadastro**:
   - Deve pedir: Nome, Idade e Salário.
   - **REGRAS**: 
     - Use `try/except` para garantir que idade e salário sejam números.
     - Se a idade for menor que 14, o sistema deve avisar: "Cadastro não permitido para menores de 14 anos." e não salvar.
3. **Relatórios**:
   - Ao final, deve usar funções como `sum()` e `len()` em uma lista (ou apenas acumuladores) para mostrar:
     - Quantidade total de pessoas cadastradas.
     - Média salarial da equipe.
4. **Organização**:
   - Use pelo menos duas funções (`def`) para organizar o código (ex: `limpar_tela()`, `calcular_media()`, ou `mostrar_menu()`).
